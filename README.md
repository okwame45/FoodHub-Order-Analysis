# FoodHub Order Analysis Project

## Overview
This project analyzes FoodHub order data to uncover insights into customer behavior, restaurant performance, and delivery efficiency. The goal is to generate actionable recommendations that can help improve business operations and customer satisfaction.

---

## Project Objectives
- Understand customer ordering patterns
- Identify peak order times
- Analyze popular cuisines and restaurants
- Evaluate delivery time and its impact on ratings
- Provide data-driven business recommendations

---

## Tools & Technologies
- Python
- Pandas & NumPy
- Matplotlib & Seaborn
- Jupyter Notebook

##  Project Phases

### 1. Data Importation
- Loaded the dataset and examined its structure
- Checked number of rows, columns, and data types

### 2. Exploratory Data Analysis (EDA)
- Explored order distribution across time
- Analyzed customer ratings and delivery times
- Examined restaurant and cuisine popularity

### 3. Data Cleaning
- Handled missing values
- Converted data types where necessary
- Ensured data consistency

### 4. Feature Analysis
- Investigated relationships between:
  - Delivery time and ratings
  - Cuisine type and order frequency
  - Order cost and customer behavior

### 5. Data Visualization
- Created visualizations to identify trends and patterns
- Used bar charts, histograms, and boxplots for insights

### 6. Insights Generation
- Extracted meaningful patterns from the data
- Interpreted results in a business context

### 7. Summary
- Consolidated key findings and recommendations

---

## Key Insights
- Peak order demand occurs during evening hours (dinner time)
- A few cuisines dominate the majority of customer orders
- Faster delivery times are associated with higher customer ratings
- Some restaurants consistently receive more orders than others
- High-value orders contribute significantly to overall revenue

---

## Business Recommendations
- Increase staffing during peak evening hours
- Partner more with high-demand restaurants and cuisines
- Improve delivery logistics to reduce delivery time
- Offer promotions during off-peak hours to balance demand
